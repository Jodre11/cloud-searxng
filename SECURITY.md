# Security Policy

## Reporting a Vulnerability

If you spot a security issue in this repository's configuration, scripts, or
documented patterns, please report it privately via
[GitHub Security Advisories](https://github.com/Jodre11/cloud-searxng/security/advisories/new)
or by emailing jodre11.public@gmail.com.

Please don't open a public issue for security concerns.

## Scope

This repository describes infrastructure that runs in **my** Oracle Cloud
tenancy. Reports about the design — credential handling, network exposure,
container hardening, IaC patterns — are welcome.

Reports about specific instances I operate (including any live deployment
reachable via my Tailnet) are out of scope. Reaching that infrastructure is
authorised only for me.
